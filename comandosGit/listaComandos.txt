git clone:<Comando utilizado para clonar (copiar, descargar) un repositorio remoto desde un servicio como GitHub a nuestra maquina Local>

git commit: <este comando captura o guarda los cambios preparados en el proyecto>

git push: <Comando utilizado para subir los commits desde un repositorio Local a un repositorio Remoto>

git status: <Comando utilizado para ver el estado actual del area de trabajo y del area de preparación (staging)>

git add: <Este comando nos ayuda a mover los cambios de nuestro proyecto al area de preparación conocido como staging area, asi preparándolos para ser incluidos en el próximo commit>
